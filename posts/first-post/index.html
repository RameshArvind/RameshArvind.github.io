Why even blog?

So then this is probably going to be a pretty boring intro to what I’m trying to get at by blogging.

I spend a lot time on the internet, it’s actually more surprising when I don’t find myself on it. I’m easily distracted when I’m trying to be productive and have a hard time trying to focus. There are specific things I’d like to achieve by blogging

  • It isn’t really an easy thing to do. That way if I’m trying to get a blog post going, I’ll stick with it till I’m done.
  • It is an easier way for me to showcase what I’m capable of. I’m pretty sure it’ll get my foot into a lot of opportunties in the future.
  • This one is probably the most important one. I’m actively trying to get into machine learning, in a more ambitious sense (Think AlphaGo). I’m taking up a bunch of courses on the path to it. Best way for me to remember things is to blog the notes I’m taking on them.

In the end If I’m probably the only guy reading this blog, I’ll still try to keep it as a useful reference for what I’m learning up on.

What you can expect from this blog?

I’ll mostly be blogging in the form of notes for the courses I’m taking. If you’re interested in machine learning you can try to make use of my notes. I will try to blog in a more structured way like a tutorial on some of the topics I’ve learned with an accompanying Jupyter notebook.
